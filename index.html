Desky

This application was developed at ICHack 2016. It is basically an android application which is compatible with all the VR headsets. The purpose of Desky is to stream all your linux desktop workspaces in 3D so that you can see all of them simultaneously. You have the option to focus on a particular screen and zoom in. This application was built using Unity with the backend server written in node.js

P.S. This is my favourite project.

Buit at my first hackathon HackKing's 2015. Mymos is a dynamic group chat application.
